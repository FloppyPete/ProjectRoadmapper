### Errors Encountered & Fixed

**1. Windows Console Encoding Error**
- **Error:** `'charmap' codec can't encode character '\U0001f4ca'`
- **Cause:** Windows PowerShell defaults to code page that doesn't support Unicode emojis
- **Fix:** Added `ensure_utf8_console()` utility function that reconfigures stdout/stderr to UTF-8
- **Prevention:** Created `roadmapper/utils.py` with UTF-8 handling utilities

**2. pyproject.toml Validation Error**
- **Error:** `project.authors[0].email` must be idn-email, given value: ""
- **Cause:** Empty email field not allowed in pyproject.toml spec
- **Fix:** Removed email field from authors (only name required)
- **Prevention:** Documented in TROUBLESHOOTING.md

**3. File Encoding Errors**
- **Error:** `'charmap' codec can't decode byte 0x9d` when reading template files
- **Cause:** File reads defaulting to system code page instead of UTF-8
- **Fix:** Created `read_text_file()` and `write_text_file()` utilities that always use UTF-8
- **Prevention:** All file operations now use utility functions consistently

**All errors fixed and prevention measures implemented:**
